# basic idea
##是否和盘口数量与价格有关
先假设每一个500ms的切片之发生了一次交易
定义：
1. 多开：空头与多头同时开仓，以多头报价成交（以对手的卖价成交，价格上涨或不变，持仓量增加）
2. 空平：价格上涨或不变，持仓量减少
3. 双开：价格不管，持仓量增加
4. 多换：价格不管，持仓量不变
5. 空换：价格不管，持仓量不变 
6. 双平：价格个下降或不变，持仓量减少
7. 多平：价格下降，持仓量减少==成交手数
8. 空开：价格下降，持仓量增加，成交数量大于0
